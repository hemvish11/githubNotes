Git hub Important Commands

Commands Explanation
git init initializes the empty repo
git add text.txt adds the text.txt file in current repo for commiting later
git status shows changes to be commited and untraced files
git status -uno displays the status of your working directory and staging area while ignoring the untracked files
git log displays the commit history of the current branch
git commit -m "first-commit" Adds the commit for the added files
git branch shows active branch
git branch -a shows all branches
git branch hemant create new branch named hemant
git checkout hemant switch to branch hemant
git checkout -b dummy create new branch named dummy then switch to new branch named dummy
--------------------- ---------------------
step#1. git checkout hemant go to the branch you want to merge
step#2. git pull https://github.com/git/git hemant pulls the branch into hemant branch
step#3. git rebase master adds the hemant(curr) branch to the master branch
--------------------- ---------------------
git push origin main Push the branch (and its commits) to remote server
git push origin main -f If you had pushed branch before,then this pushes the branch (and its commits) to remote server forcefully with newer commit
git pull origin main Pull the branch (and its commits) from remote server (here,must be on main branch)
git merge main This command merges the specified(main) branch into the current branch.
git reset --hard undo all your changes in the branch(locally)
git reset --hard origin/main undo all your changes in the branch (remotely)
git checkout -- text2.txt undo changes in text2.txt file
git rev-list --count HEAD ^master This command will count the number of commits that are unique to the current branch compared to the master branch.
git reset --soft head~5 prepares your last 5 commits to be squash into one. To be followed by git commit.
git commit --amend Change last commit msg
git stash save Saves uncommitted changes into a stack
git stash save "This is my message Saves uncommitted changes into a stack( with a message)
git stash list Lists stash stack
git stash pop Brings back changes (uncommitted) saved at top of stash stack.
